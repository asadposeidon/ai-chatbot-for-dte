# AI Chatbot for DTE

This project is a modern landing page for an AI-powered chatbot for the Department of Technical Education (DTE).  
It mirrors a Figma design for a single-page site that explains and demonstrates the chatbot.

## Features

- Fixed navigation bar with primary **Start Chat** call-to-action
- Hero section describing the chatbot and its purpose
- About, Features, How It Works, Chat Demo, Benefits, and Technology Stack sections
- Mock chatbot interface showcasing example conversations
- Responsive layout using plain HTML and CSS

## Tech Stack

- HTML5
- CSS3
- Google Fonts (`Inter`)

## Getting Started

1. Open `index.html` directly in your browser, or serve the folder using a static server.
2. Edit `styles.css` to customize colors, spacing, or typography if needed.
